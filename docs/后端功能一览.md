## 二、认证与用户隔离

### 认证机制

- **JWT (JSON Web Token)** — 使用 `python-jose` 签发
- **密码加密** — `bcrypt`（通过 `passlib`）
- **Token 存储** — 前端（Web: localStorage / 小程序: wx.setStorageSync）
- **Token 有效期** — 默认 30 分钟（可配置）

### 登录方式

| 方式 | 接口 | 说明 |
|------|------|------|
| 用户名+密码 | `POST /auth/login` | 标准账号密码登录，返回 JWT |
| 注册 | `POST /auth/register` | 创建新用户 |
| 微信小程序 | `POST /auth/wx-login` | 微信 code → 换 openid → 自动注册/登录 |
| 当前用户 | `GET /auth/me` | 验证 Token 有效性，返回用户信息（需 `Authorization: Bearer`） |

### 微信小程序登录流程

1. 前端调用 `Taro.login()` 获取临时 code
2. 发送 code 到 `POST /auth/wx-login`
3. 后端通过 `https://api.weixin.qq.com/sns/jscode2session` 换取 openid
4. 查表：openid 已存在 → 直接登录；不存在 → 创建新用户
5. 返回 JWT + `has_phone` 标记
6. **开发/测试模式**：阿里云 AppID/Secret 未配置时，自动模拟 openid（`mock_openid_***`）

### 手机号绑定

- **验证码发送**：`POST /auth/send-code`（阿里云短信 SDK）
- **绑定**：`POST /auth/bind-phone`（验证码校验 + 手机号唯一性检查）
- **开发/测试模式**：未配置阿里云短信时，验证码直接返回在响应中

## 三、行情模块 `/market`

### 股票数据
- **`GET /stocks/search`** — 按代码/名称模糊搜索股票
- **`GET /stocks/{id}/prices`** — 获取日 K 线（可选天数 1-365）
- **`GET /stocks/{id}/realtime`** — 实时概览（最新价 + 涨跌幅 + 成交量）
- **`GET /stocks/all`** — 全量股票分页列表（含最新价、涨跌幅、基本面）
- **`POST /stocks/load-all`** — 从 AKShare 批量注入 A 股列表
- **`POST /stocks/{id}/fetch-realtime`** — 拉取单只股票真实 K 线（AKShare）
- **`POST /stocks/{id}/fetch-fundamentals`** — 拉取基本面数据
- **`POST /stocks/fetch-all-realtime`** — 刷新所有自选股 K 线

### 大盘指数
- **`GET /indices`** — 从 A 股 price 聚合计算上证/深证/创业板模拟指数

### 自选股（需用户隔离）
- **`GET /watchlists`** — 获取当前用户的所有自选股分组（含股票列表）
- **`POST /watchlists`** — 创建分组
- **`PATCH /watchlists/{id}`** — 重命名分组
- **`DELETE /watchlists/{id}`** — 删除分组
- **`POST /watchlists/{id}/items`** — 添加股票到分组
- **`DELETE /watchlists/{id}/items/{stock_id}`** — 从分组移除股票

### 交易日历
- **`GET /trading-status`** — 判断 A/港股/美股当前交易状态（盘前/交易中/盘后/休市）

## 四、分析模块 `/analysis`

### 技术指标
- **`GET /stocks/{id}/indicators`** — 计算并返回五类技术指标：
  - MA (5/10/20/60)
  - MACD (DIF/DEA/MACD)
  - RSI14
  - BOLL (上/中/下轨)
  - KDJ (K/D/J)
- **`POST /stocks/{id}/indicators`** — 批量计算并存储到数据库

### 形态识别
- **`GET /stocks/{id}/patterns`** — 识别双底、头肩顶等 K 线形态

### 综合分析
- **`GET /stocks/{id}/comprehensive`** — 综合仪表盘（指标 + 形态 + 趋势评分 + 基本面）

### 行业板块
- **`GET /sectors`** — 按 sector 分组统计（数量、平均PE/PB、平均涨跌幅、成分股）

> 分析模块为纯计算服务，不涉及用户数据。

## 五、策略模块 `/strategies`（需用户隔离）

- **`GET /`** — 列出当前用户的所有策略
- **`POST /`** — 创建策略（定义条件表达式 + 动作）
- **`GET /{id}`** — 查看单个策略
- **`PATCH /{id}`** — 更新策略
- **`DELETE /{id}`** — 删除策略
- **`GET /{id}/triggers`** — 获取策略触发记录
- **`POST /{id}/test-trigger`** — 手动触发测试（模拟条件评估 + 推送）
- **`GET /templates/all`** — 获取内置策略模板列表
- **`POST /templates`** — 初始化 5 个内置模板

### 支持的条件类型
| 条件类型 | 说明 |
|---------|------|
| `price_above` | 价格超过阈值 |
| `price_below` | 价格跌破阈值 |
| `price_change_pct` | 涨跌幅超过阈值 |
| `volume_surge` | 成交量异常放大 |
| `ma_cross` | 均线金叉/死叉 |
| `macd_cross` | MACD 金叉/死叉 |
| `rsi_threshold` | RSI 超买/超卖 |
| `bollinger_break` | 布林带突破 |
| `trend_score` | 趋势评分阈值 |
| `ai_judge` | AI 自定义条件判断 |
| `always` | 始终触发（用于定时推送） |

## 七、AI 分析模块 `/ai`

- **`GET /summary/{stock_id}`** — AI 个股多维度分析摘要（趋势/支撑/阻力/风险/评分）
- **`POST /query`** — 自然语言查询
- **`POST /digest`** — 生成每日智能简报
- **`POST /predict/{stock_id}`** — AI 个股涨跌预测（方向/幅度/置信度/支撑阻力）

### AI 配置（需用户隔离）
- **`GET /settings`** — 获取当前用户 AI 配置
- **`POST /settings`** — 保存 AI 配置（支持 OpenAI/DeepSeek/Together）
- **`DELETE /settings/{id}`** — 删除配置

AI 不可用时自动降级为规则兜底（简单均线 + RSI 判断）。

### AI 选股（需用户隔离 — 创建账户归到当前用户）
- **`POST /ai-generate`** — AI 选股生成投资组合建议
- **`POST /ai-create`** — AI 选股并直接创建模拟账户+买入
- **`POST /ai-auto-pick`** — 定时 AI 全市场选股 Top N + 自动组合创建

## 八、模拟交易模块 `/portfolio`（需用户隔离）

### 账户管理
- **`POST /accounts`** — 创建模拟账户
- **`GET /accounts`** — 列出当前用户的所有账户（含持仓汇总、盈亏）

- **`POST /accounts/{id}/withdraw`** — 提现

### 交易
- **`POST /accounts/{id}/trades`** — 买入/卖出（支持市价单/限价单）
- **`GET /accounts/{id}/trades`** — 交易流水

### 费用模型
- 佣金率: A 股万1.5 / 港股万2.5 / 美股万1 （可配置）
- A 股印花税: 卖出万5
- 滑点: 可配置（默认 0.1%）
