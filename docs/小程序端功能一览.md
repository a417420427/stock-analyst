## 二、认证体系

### 登录方式

| 方式 | 适用场景 |
|------|---------|
| **微信一键登录** | 主要登录方式，调用 `Taro.login()` → 后端换 openid |
| **用户名+密码登录** | 备用登录方式（`POST /auth/login`） |
| **注册** | `POST /auth/register` 创建新用户 |

### 微信登录流程

```
用户点击"微信登录"
    ↓
Taro.login() → 获取临时 code
    ↓
POST /auth/wx-login?code=xxx （无 auth header，来源匿名）
    ↓
后端 → https://api.weixin.qq.com/sns/jscode2session → openid
    ↓
查 users 表：openid 已存在？是→登录；否→创建新用户
    ↓
返回 JWT token + 用户信息 + has_phone 标记
    ↓
前端存 Token 到 Taro.setStorageSync('stock_token')
    ↓
后续所有请求自动带上 Authorization: Bearer
```

### 手机号绑定

> 微信登录后如果 `has_phone == false`，提示绑定手机号。

```
输入手机号
    ↓
POST /auth/send-code?phone=xxx（需登录, 带 Token）
    ↓
阿里云短信 → 用户收到验证码
开发模式: 验证码直接返回在响应中（debug_code）
    ↓
用户输入验证码
    ↓
POST /auth/bind-phone?phone=xxx&code=xxx（需登录）
    ↓
验证码校验成功 → phone 写入 users.phone
```

### Token 管理

| 操作 | 方式 |
|------|------|
| 存储 | `Taro.setStorageSync('stock_token', token)` |
| 读取 | `Taro.getStorageSync('stock_token')` |
| 清除 | `Taro.removeStorageSync('stock_token')` |
| 请求携带 | `Authorization: Bearer xxx` |
| 401 处理 | 清除 Token → `Taro.navigateTo('/pages/login/index')` |
| 用户信息 | `Taro.setStorageSync('stock_user', user)` |

### ✅ 用户隔离

> 所有业务 API 均已按当前登录用户隔离数据，每个用户只能看到自己的自选股、策略、组合和日志。

## 三、页面功能详解

### 1. 大盘页 `pages/index/index`
- **大盘指数**: 展示上证指数、深证成指、创业板指的模拟走势
- **K 线趋势图**: 每个指数展示最近 60 个交易日的收盘价曲线
- **涨跌幅标识**: 绿色下跌、红色上涨
- 调用 `GET /market/indices`（公开接口，无需登录）

### 2. 股票页 `pages/stocks/index`
- **搜股票**: 按代码或名称模糊搜索（支持 A/HK/US 三市场）
- **市场筛选**: 可切换 A 股 / 港股 / 美股
- **股票列表**: 代码 + 名称 + 市场 + 最新价 + 涨跌幅
- 调用 `GET /market/stocks/search`, `GET /market/stocks/all`（公开接口）

### 3. 个股详情页 `pages/stock-detail/index`
- **实时概览**: 股票名称、代码、市场、最新价、涨跌幅、成交量、更新时间
- **K 线图**: 最近 60 天日线图（折线图，显示开盘/收盘价范围）
- **技术指标**: MA5/MA20 均线（叠加在 K 线上方）
- **综合信息**: PE(TTM)、PB、市值、行业
- **AI 摘要**: 调用 AI 生成个股分析摘要（趋势判断/支撑/阻力/风险/评分）
- **AI 预测**: 调用 AI 预测涨跌方向与幅度
- 调用:
  - `GET /market/stocks/{id}/realtime`（公开）
  - `GET /market/stocks/{id}/prices`（公开）
  - `GET /analysis/stocks/{id}/comprehensive`（公开）
  - `GET /ai/summary/{id}`（公开）
  - `POST /ai/predict/{id}`（公开）

### 4. 组合页 `pages/portfolio/index`（需登录）
- **账户列表**: 展示当前用户的所有模拟交易账户
- **每个账户卡片**:
  - 账户名称
  - 总资产 / 初始资金
  - 总盈亏（金额 + 百分比）
  - 持仓数量
  - 市场价值
  - AI 组合标识（`is_ai_generated`）
- 调用 `GET /portfolio/accounts`

### 5. 板块页 `pages/sectors/index`
- **行业板块列表**: 按行业名称分组
- **每个板块卡片**:
  - 行业名称 + 股票数量
  - 平均 PE / 平均 PB
  - 平均涨跌幅
  - 成分股预览（前几支股票代码 + 涨跌幅）
- 调用 `GET /analysis/sectors`（公开接口）

### 6. 登录页 `pages/login/index`
- **微信一键登录**: 入口按钮 → 调用 wxLogin API
- **用户名密码登录**: 表单输入
- **注册**: 新用户注册表单
- **手机号绑定**: 发送验证码 → 输入验证码 → 绑定
- **退出登录**: 清除本地 Token 和用户信息
- 调用:
  - `POST /auth/wx-login`（公开）
  - `POST /auth/login`（公开）
  - `POST /auth/register`（公开）
  - `POST /auth/send-code`（需登录）
  - `POST /auth/bind-phone`（需登录）

### 7. 日志页 `pages/logs/index`（需登录）
- **操作日志列表**: 按时间倒序展示
  - AI 选股记录
  - 交易记录
  - 系统通知
- 调用 `GET /logs`

## 四、API 调用明细

### 公开接口（无需 Token）

| API | 用途 | 页面 |
|-----|------|------|
| `GET /market/indices` | 大盘指数走势 | 大盘 |
| `GET /market/stocks/search` | 搜索股票 | 股票 |
| `GET /market/stocks/all` | 股票列表 | 股票 |
| `GET /market/stocks/{id}/realtime` | 个股实时 | 个股详情 |
| `GET /market/stocks/{id}/prices` | K 线数据 | 个股详情 |
| `GET /analysis/stocks/{id}/comprehensive` | 综合分析 | 个股详情 |
| `GET /analysis/sectors` | 行业板块 | 板块 |
| `GET /ai/summary/{id}` | AI 分析摘要 | 个股详情 |
| `POST /ai/predict/{id}` | AI 涨跌预测 | 个股详情 |
| `POST /auth/login` | 账号登录 | 登录 |
| `POST /auth/register` | 注册 | 登录 |
| `POST /auth/wx-login` | 微信登录 | 登录 |

## 六、类型定义

`src/types/index.ts` 定义了所有数据类型的 TypeScript 接口:

| 接口 | 用途 |
|------|------|
| `Stock` | 股票基础信息（id/symbol/market/name/基本面） |
| `PriceData` | 日 K 线数据（date/open/high/low/close/volume） |
| `StockRealtime` | 实时行情（price/change/volume/date） |
| `IndexData` | 大盘指数（name/prices/change_pct） |
| `ComprehensiveAnalysis` | 综合分析结果（指标+形态+趋势+基本面） |
| `AIAnalysisResponse` | AI 分析摘要 |
| `AIPredictionResponse` | AI 涨跌预测 |
| `SimulatedAccount` | 模拟账户（资产/盈亏/策略关联） |
| `SectorData` | 行业板块（统计+成分股） |
| `ActivityLog` | 操作日志 |
| `User` | 用户信息 |
